# M1 任务清单 —— 单人完整版

> 目标：一个能**搜索基金 → 加自选 → 录持仓 → 看涨幅盈亏**的可运行单人应用。
> 开发期用种子数据；部署到联网环境后一键同步全市场 1.8 万只。

## 数据层

- [ ] `backend/models/db.py` —— SQLite 建表：`fund_list` / `fund_quote` / `holding`
- [ ] 种子数据：几十只常见基金写入 `fund_list`（含代码/名称/拼音/类型）

## 抓取层（backend/datasource/）

- [ ] `fundgz.py` —— 盘中估值（gsz/gszzl），基于已验证的 spike 脚本
- [ ] `fund_list_sync.py` —— 全量列表同步脚本（沙箱受限，部署环境运行）
- [ ] `akshare_nav.py` —— 收盘官方净值（M1 可先留桩，M2 完善）

## 业务 API（backend/app.py）

- [ ] `GET /api/search?q=` —— 查本地 fund_list，返回匹配基金
- [ ] `POST /api/holdings` —— 加自选 / 录持仓（代码+持仓金额+成本+预期）
- [ ] `GET /api/holdings` —— 我的自选列表 + 实时估值 + 盈亏 + 距预期差距
- [ ] `DELETE /api/holdings/{id}` —— 移除自选

## 前端（frontend/）

- [ ] 搜索框 + 结果下拉 → 点击加自选
- [ ] 自选卡片：名称 / 当日涨幅 / 持仓金额 / 今日盈亏 / 距目标差距
- [ ] 录入持仓 & 预期的表单

## 验收标准

1. 搜索框输入"机器人"或"020608"，能搜到基金
2. 点击加入自选，填持仓金额
3. 列表实时显示当日涨幅（gszzl）和今日浮动盈亏
4. 全程读本地库 + 缓存，不高频打外部接口
