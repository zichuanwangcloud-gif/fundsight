# 盈见 FundSight —— 立项文档

> 版本：v1.0　｜　日期：2026-07-02　｜　状态：立项通过（GO）

## 一句话定位

一个网页版的自选基金看板：录入基金、持仓成本和**预期目标**，每天打开一眼看到
**实际涨幅、真实盈亏、以及离我的目标还差多少**，触发止盈/止损时提醒我。

## 为什么要自己做（差异点）

天天基金、支付宝、雪球都能看涨幅和持仓收益。盈见的差异只有一句话：

> **它们展示"现状"，盈见展示"现状 vs 我的预期"。**

- 别的 App 给你看**实际**涨幅 / 收益
- 盈见让你先设定**预期**（目标收益率、目标价、止盈止损线），每天看**实际与预期的差距**，辅助决策

这条是产品的魂。开发过程中若发现自己在做"又一个净值查询器"，就是跑偏了。

## 目标用户

- 主要：**自己用**
- 次要：**分享给几个朋友**（少量多用户，需数据隔离，但**不做对外正式运营**）

> ⚠️ 合规红线：一旦未来公开注册 / 收费，性质就从"自用私享"变为"对外提供服务"，
> 数据合规风险陡增（见架构文档「合规边界」）。当前形态刻意止步于私享。

## MVP 功能范围（第一版只做这些）

| 模块 | 内容 | 优先级 |
|---|---|---|
| **基金搜索** | 搜全市场基金（代码/名称/拼音），查本地全量列表 | P0 |
| 自选列表 | 从搜索结果加入/移除自选、展示当日涨跌幅 | P0 |
| 持仓录入 | 成本价 / 份额（或持仓金额）→ 算真实盈亏 | P0 |
| 预期设定 | 目标收益率或目标价 → 算差距 | P0 |
| 止盈止损提醒 | 触发线时页面高亮 / 角标（先不做推送） | P1 |
| 多用户 | 简单登录 + 数据隔离（自用 + 朋友） | P1 |

> **范围说明（2026-07-02 升级）**：产品是一个**完整应用**，不是只看预设的几只。
> 用户自己搜索全市场基金 → 加自选 → 录持仓。搜索走**本地全量列表**（约 1.8 万只，
> 后台低频同步），不实时打外部接口——见架构文档「搜索设计」。

### 刻意不做（Out of Scope，防止范围膨胀）

手机推送、K 线图、大盘分析、收益走势历史图、社区/评论、基金推荐。
以上均为 P2 之后，第一版一律不碰。

## 最大风险及其结论

**唯一能让项目直接黄掉的点是数据来源。已于 2026-07-02 实测排除。**

- 用基金编号 `020608`（南方中证机器人ETF发起联接C）实测 `fundgz.1234567.com.cn` 接口，
  成功拿到名称、当日估算涨幅（gszzl）、盘中估值（gsz）、昨日净值（dwjz）。
- 纯 Python 标准库、零依赖、免费。
- 验证脚本见 `scripts/fund_spike.py`，运行结果见 `docs/02-数据源调研.md`。

**GO / NO-GO 判断：GO。**
