# 盈见 FundSight

> 预见收益 —— 一个自用的网页版基金看板

天天基金给你看**现状**，盈见让你看**现状 vs 你的预期**。
录入自选基金、持仓和预期目标，每天一眼看到：当日涨幅、真实盈亏、离目标还差多少。

## 快速验证数据链路

```bash
python3 scripts/fund_spike.py
```

会拉取一只真实基金，打印当日涨幅、盘中估值，并演示盈亏计算。

## 技术选型

Python（数据 + API）· SQLite（缓存 + 存储）· 单页前端 · 先本地跑
